# Example Module

This is a simple module README to test multi-file conversion.

## Installation

```bash
npm install example-module
```

## Usage

```javascript
const example = require('example-module');
example.doSomething("hello");
```

## API

| Method | Description | Returns |
|--------|-------------|---------|
| `doSomething(input)` | Processes input string | `string` |
| `reset()` | Resets internal state | `void` |

## Notes

> [!TIP]
> This module works best with Node.js 18+.

See the [main project README](../README.md) for more details.
